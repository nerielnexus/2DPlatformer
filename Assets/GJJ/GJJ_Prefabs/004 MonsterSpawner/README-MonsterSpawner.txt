< GJJ_MonsterSpawner 메뉴얼 >

특정 조건을 만족하면 정해진 몬스터를 특정 위치에 생성하거나 전부 Destroy 하는 몬스터 스포너이다.

1. Monster To Spawn
    - 몬스터 스포너를 통해 생성할 몬스터의 Prefab 을 설정한다.

2. Monster Spawn Interval
    - 하나의 몬스터를 생성한 후, 얼마만큼 대기해야 다시 생성할 수 있는지 설정한다.

3. Monster Max Spawn Count
    - 하나의 몬스터 스포너가 생성할 수 있는 몬스터의 최대 수를 설정한다.

4. Readonly
    - 몬스터가 생성되는 위치의 목록 (Monster Spawn Point List)
    - 생성된 몬스터의 목록 (Monster Spawned List)

5. 현재 조건
    - 키보드 M 키로 몬스터 생성
    - 키보드 N 키로 생성된 몬스터 중 하나를 제거
