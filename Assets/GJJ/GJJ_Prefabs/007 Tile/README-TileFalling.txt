< GJJ_TileFalling 메뉴얼 >

1. Tile Falling Type
    - 플레이어 캐릭터가 타일 위에 올라탔을 때, 타일이 떨어지는지를 설정한다.
    - NOFALL 은 타일이 떨어지지 않는다.
    - NORESPAWN 은 타일이 떨어지며, 이후 다시 생성되지 않는다.
    - CANRESPAWN 은 타일이 떨어진 이후, 일정 시간이 지나고 다시 생성된다.

2. Tile Falling Time Delay
    - 플레이어가 타일 위에 올라타고, 타일이 떨어지기까지의 대기 시간을 설정한다.

3. Tile Falling Time
    - 떨어지는 타일이 몇 초 뒤에 사라질지 설정한다.
